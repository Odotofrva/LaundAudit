# Laundromat Growth Dashboard

Open `index.html` in your browser.

What it does:
- Tracks daily shift metrics
- Tracks machine issues and downtime
- Tracks customer feedback and complaints
- Tracks inventory/supplies
- Shows KPI cards, charts, and recommendations
- Includes a 30/60/90 day progress plan
- Saves data in your browser with localStorage
- Exports JSON backup and daily metrics CSV

Important:
This starter version has no backend or login. Data is stored in the browser you use.
Use Export Data regularly if you want backups.
